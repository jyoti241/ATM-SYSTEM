#  ATM Management System

##  Project Overview

ATM Management System is a Java-based banking application that simulates the core functionalities of an Automated Teller Machine (ATM). The project is designed to provide users with a secure and user-friendly interface for performing basic banking operations such as account authentication, balance inquiry, cash withdrawal, cash deposit, PIN management, and transaction tracking.
This project demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts, user input handling, and banking transaction management using Java.

---

##  Features

###  User Authentication
- Secure login using account number and PIN.
- Access control for authorized users only.

###  Balance Inquiry
- View current account balance instantly.

###  Cash Withdrawal
- Withdraw available funds from the account.
- Validation for sufficient balance.

###  Cash Deposit
- Deposit money into the account.
- Automatic balance updates.

###  PIN Change
- Change account PIN securely.

###  Transaction History
- View details of previous transactions.

###  Exit System
- Safely terminate the application session.

##  Installation & Setup

### Prerequisites

Before running the project, ensure you have:

- Java JDK 8
- IntelliJ IDEA, Eclipse,


### Clone the Repository


git clone https://github.com/jyoti24/ATM-System.git


### Open the Project

1. Open IntelliJ IDEA or Eclipse.
2. Select **Open Project**.
3. Navigate to the ATM-System folder.
4. Import the project.

### Run the Application

Compile and execute the main Java file:


javac Main.java
